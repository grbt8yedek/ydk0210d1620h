## 🚀 Geri Yükleme

```bash
# 1. Repo'yu clone et
git clone https://github.com/grbt8yedek/ydk0210d1620h.git

# 2. Bağımlılıkları yükle
npm install

# 3. .env dosyasını oluştur
cp .env.example .env.local
# .env.local dosyasını düzenle!

# 4. Prisma'yı hazırla
npx prisma generate
npx prisma db push

# 5. Geliştirme sunucusunu başlat
npm run dev
```
